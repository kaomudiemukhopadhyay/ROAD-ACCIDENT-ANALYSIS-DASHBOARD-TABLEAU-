📌 Project Overview
This project presents an interactive Tableau dashboard to analyze road accident data across multiple dimensions such as accident severity, weather conditions, road types, and geographical distribution.
The dashboard enables stakeholders to identify patterns, monitor trends, and make data-driven decisions to improve road safety.
Dashboards are widely used to combine multiple visual insights into one interface for better decision-making

🎯 Objective
The main objectives of this project are:
Analyze total accidents and casualties
Understand fatal, serious, and slight accident trends
Identify high-risk factors (weather, road surface, vehicle type)
Explore geographical distribution of accidents
Enable interactive filtering using parameters

## ⚙️ Advanced Tableau Concepts Used
### ✅ Calculated Fields
Created custom calculations to derive meaningful insights, including:
* Year-over-Year (YoY) growth analysis
* Total accidents and casualty counts
* Ratio-based KPIs for performance comparison

### ✅ Parameters
Implemented dynamic parameters to enhance user control and flexibility:
* Current Year vs Previous Year selection
* Accident Severity selection (Fatal, Serious, Slight)

### ✅ Filters
Applied multiple types of filters for efficient data exploration:
* Quick Filters for user interaction
* Context Filters to improve performance
* Dimension & Measure Filters for precise data control

### ✅ Grouping
Grouped similar categories to simplify analysis and improve readability:
* Combined road types and categories for better comparison
* Reduced data complexity for clearer insights

### ✅ Sets
Used sets to highlight and analyze specific segments:
* Identified high-risk categories and top contributing factors
* Enabled comparison between selected and non-selected data

### ✅ Hierarchies
Built hierarchies to allow drill-down analysis:
* Date hierarchy (Year → Month) for trend analysis
* Improved navigation across different levels of data

### ✅ Dashboard Actions
Implemented interactive actions to enhance user experience:
* Filter Actions for dynamic chart interaction
* Highlight Actions for better data focus

### ✅ Dashboard Design
Designed a clean and professional dashboard layout using:
* **Tiled Layout** for structured and aligned visuals
* **Floating Layout** for flexible placement of filters and control panels

### ✅ Data Visualization Best Practices
* Used appropriate charts (bar, donut, map) for clarity
* Maintained consistent color themes and formatting
* Focused on readability and user experience



## 💡 Key Insights
* Majority of accidents occur under **dry road conditions**, indicating driver behavior plays a key role
* **Cars contribute the highest number of fatal casualties**, making them the most critical vehicle category
* **Single carriageway roads** are the most accident-prone, highlighting infrastructure risks
* Certain regions show **high accident density**, indicating potential accident hotspots
* Monthly trends reveal **seasonal variations**, with noticeable fluctuations across the year
